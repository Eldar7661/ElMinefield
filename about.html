Minesweeper

Проект Сапёр(Пародия), был сделан студентом, 4-го курса группы ИС_201 коледжа АГКЭиЭТ(KZ/Алматы), Алиев.Э.

Данный проект это курсовая работа на 4-ом курсе, про (ООП) на произвольную тему. Игра написана на Python 3.11.6 (PyQt5).

Горячие клавиши:

  • Ctrl+Q : Exit
  • Ctrl+R : Restart
  • Ctrl+N : New level
  • Ctrl+S : Settings
  • Ctrl+H : About
